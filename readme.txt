JNI interface for Epiphan frame grabbers. Java 1.4 or better is required.

Contents of this distribution:

  vga2usb.jar      - compiled interface package (com.epiphan.vga2usb)
  v2ujni.dll       - native JNI library for Windows (32-bit)
  libv2ujni.jnilib - native JNI library for Mac OS X (i386,ppc,x86_64)
  libv2ujni.so     - native JNI library for Linux (i386)
  test.jar         - test application

Directories:

  doc              - Javadoc documentation
  src              - Source code for test and interface packages
  64-bit           - Native 64-bit binaries for Linux and Windows

Type 'java -jar test.jar' to run the test.
